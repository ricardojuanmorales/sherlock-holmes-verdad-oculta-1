# Rúbrica de Evaluación — Sherlock Holmes y la Verdad Oculta V1.0

> Documento de uso exclusivo para Game Master y mentores.

---

## Dimensiones de evaluación (100 puntos base)

| Dimensión | Peso | Criterio principal |
|---|---|---|
| 1. Rigor investigativo | 25 pts | Evidencias sólidas · triangulación · hipótesis contrastadas |
| 2. Calidad documental | 25 pts | Expediente completo · cadena de custodia · bitácoras al día |
| 3. Calidad técnica del MVP | 20 pts | Funcionalidad · claridad · relación con el caso |
| 4. Uso ético de IA | 15 pts | Transparencia · validación humana · IA-log completo |
| 5. Trabajo en equipo | 15 pts | Roles definidos · coordinación · Quinto Elemento activo |

## Multiplicador documental (bonificación)

| Nivel documental | Multiplicador |
|---|---|
| Básico pero válido | +5% |
| Claro y consistente | +10% |
| Muy sólido y trazable | +15% |
| Ejemplar y auditado | +20% |

## Faltas que restan puntos

| Falta | Penalización |
|---|---|
| IA no documentada | -10 pts |
| Evidencia sin cadena de custodia | -5 pts por evidencia |
| Conclusión sin sustento | -15 pts |
| Plagio o fabricación de evidencia | Descalificación del caso |
